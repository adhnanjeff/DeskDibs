# docs/

## openapi.json

The frozen OpenAPI 3 contract for the DeskDibs backend, generated by springdoc from the real
running application — not hand-written. The frontend phases (5+) build against this file so the
contract is stable while the backend keeps moving underneath.

### Regenerating it

Whenever a controller's request/response shape changes, regenerate before merging:

```bash
cd backend
export AUTH_PROVIDER=local
mvn spring-boot:run &
sleep 5
curl -s http://localhost:8080/v3/api-docs | python3 -m json.tool > ../docs/openapi.json
kill %1
```

The dev database (`deskdibs`, from `.env`) must be reachable — the app won't boot without it.
`AUTH_PROVIDER=local` avoids needing a real Entra tenant just to start the server.

You can also browse the live contract at `http://localhost:8080/swagger-ui.html` while the app is
running, or fetch `http://localhost:8080/v3/api-docs` directly — both endpoints are permitted
unauthenticated (see `SecurityConfig.OPENAPI_PATHS`) specifically so CI and the frontend can read
them without a token.
